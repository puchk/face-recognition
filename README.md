# face-recognition
React app that uses Clarifai API to detect faces in images. 

# Live Link
 - https://face-recognit.herokuapp.com/
 
# Technologies Used
 - Node
 - React
 - PostgreSQL
 - Express
 - Bcrypt
 - Clarifai API
 - Tachyons
 - Knex
 - CORS
